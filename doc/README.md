# VX6 Website Content

This folder contains short, page-ready website copy for VX6.

Files:

- `homepage.md` - hero copy, about section, feature blocks, and homepage CTA text
- `taglines.md` - safer tagline options and claim guidance
- `beginner-guide.md` - simple introduction for people who do not know IPv6
- `who-uses-vx6.md` - use cases, audiences, and practical value
- `progress.md` - current progress and what VX6 already does
- `how-to-use.md` - short tutorial-style usage guide
- `technical-working.md` - technical explanation in readable language
- `hidden-services.md` - detailed hidden-service page copy
- `build-on-vx6.md` - builder, platform, and SDK-style positioning
- `install-download.md` - install and download page copy
- `decentralization.md` - the decentralization story and why it matters
- `vx6-vs-tor.md` - comparison copy for people evaluating VX6 against Tor-style networks
- `diagrams-ascii.md` - ASCII diagrams for website conversion

Writing rules used here:

- short
- readable
- human
- SEO-friendly
- accurate to the current VX6 model

Current release assumptions:

- Linux support is published through the `main` branch
- Windows support is published through the `Windows-compatible` branch
- both branches follow the same current protocol and feature behavior
- current transport is TCP-only
- `vx6-gui` exists now
- broader browser-wrapper work is still upcoming
